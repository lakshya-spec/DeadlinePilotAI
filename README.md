# DeadlinePilot AI

An AI-powered deadline recovery agent that helps students and professionals prioritize tasks, assess deadline risk, generate realistic action plans, and recover when they fall behind.

## Core Features
- Task management with deadlines and estimated effort
- Deadline Risk Meter
- AI daily planner powered by Gemini
- Rescue Mode for last-minute recovery plans
- Smart task breakdown and next-best-action guidance

## Tech Stack
- HTML
- CSS
- JavaScript
- Google Gemini API
- Google AI Studio
- LocalStorage
- GitHub
